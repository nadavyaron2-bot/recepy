# Recepy - אפליקציית המתכונים האישית שלך 🍳

Recepy היא אפליקציית אנדרואיד מודרנית לניהול, שמירה וייבוא של מתכונים מכל רחבי הרשת. האפליקציה מתמקדת בחוויית משתמש נקייה, תמיכה מלאה בעברית וכלים חכמים למטבח.

## ✨ תכונות עיקריות

*   **ייבוא חכם:** הדבק קישור מכל אתר (Mako, Foody, Walla ועוד) והאפליקציה תחלץ עבורך את המצרכים ואופן ההכנה באופן אוטומטי.
*   **חיפוש חכם (Mako Integration):** אם חיפשת מתכון ולא מצאת אותו באוסף האישי שלך, האפליקציה תציע לך לחפש במאקו ותטען את המתכון הראשון ישירות לתוך התצוגה של האפליקציה.
*   **ניהול מצרכים:** סמן מצרכים שכבר יש לך בזמן הבישול.
*   **רשימת קניות:** הוסף מצרכים ממתכונים ישירות לרשימת קניות מובנית.
*   **מצב מפתח:** כלי עזר למפתחים הכולל צפייה בבאגים שדווחו ומתכונים מבוקשים.
*   **ערכות נושא:** תמיכה במצב לילה (Dark Mode) ובמגוון צבעי נושא (כתום, כחול, ירוק, ורוד, סגול).
*   **גיבוי וסנכרון:** אפשרות לייצא ולייבא את כל המתכונים כקובץ JSON.

## 🛠️ טכנולוגיות

*   **Language:** Kotlin
*   **UI:** Jetpack Compose (Material 3)
*   **Database:** Room DB
*   **Parsing:** Jsoup (HTML Parsing)
*   **Concurrency:** Kotlin Coroutines & Flow
*   **Navigation:** Compose Navigation
*   **Dependency Injection:** Manual / ViewModel Factory

## 🚀 איך מתחילים?

1.  שבט את הפרויקט אליך למחשב.
2.  פתח את הפרויקט ב-Android Studio (גרסת Koala ומעלה מומלצת).
3.  הרץ את האפליקציה על אמולטור או מכשיר פיזי.
